API Referansı

API Overview

The Lydian Agent REST API provides programmatic access to all 7 AI agents, patient data, quantum optimization, and real-time monitoring. All endpoints use JSON and require OAuth 2.0 authentication.

Base URL: https://api.lydian-agent.com/v1

Current Version: v1.2.0

Authentication

Obtain Access Token

POST /api/v1/auth/token

Request:
{
  "grant_type": "client_credentials",
  "client_id": "your_client_id",
  "client_secret": "your_client_secret",
  "scope": "clinical.read clinical.write quantum.access"
}

Response:
{
  "access_token": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "Bearer",
  "expires_in": 3600,
  "scope": "clinical.read clinical.write quantum.access"
}

Include the token in all API requests:

Authorization: Bearer YOUR_ACCESS_TOKEN
