Accessibility Commitment

Lydian Agent is committed to ensuring digital accessibility for all people, including those with disabilities. We continuously work to improve the accessibility and usability of our platform to ensure equal access to healthcare AI technology.

WCAG 2.1 Compliance

Our platform conforms to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards:

  • Perceivable: All information and UI components are presented in ways users can perceive (text alternatives, captions, adaptable layouts)
  • Operable: UI components and navigation are operable by keyboard, touch, and assistive technologies
  • Understandable: Information and UI operation are clear and understandable
  • Robust: Content works across assistive technologies, browsers, and devices

Accessibility Features

Screen Reader Support

  • Semantic HTML structure with proper headings hierarchy
  • ARIA labels and landmarks for navigation
  • Descriptive alt text for all images and icons
  • Keyboard-accessible interactive elements
  • Tested with JAWS, NVDA, VoiceOver

Keyboard Navigation

  • Full keyboard operability (Tab, Arrow keys, Enter, Escape)
  • Visible focus indicators
  • Logical tab order
  • Skip navigation links
  • No keyboard traps

Visual Design

  • WCAG AA color contrast ratios (4.5:1 for text, 3:1 for UI)
  • Resizable text up to 200% without loss of functionality
  • Support for Windows High Contrast mode
  • No information conveyed by color alone
  • Clear visual hierarchy and spacing

Responsive Design

  • Mobile-first responsive layouts
  • Touch targets minimum 44x44 pixels
  • Zoom and text spacing support
  • Orientation-independent design

Assistive Technology Compatibility

Lydian Agent has been tested with:

  • Screen Readers: JAWS, NVDA, VoiceOver, TalkBack
  • Speech Recognition: Dragon NaturallySpeaking, Windows Speech Recognition
  • Screen Magnifiers: ZoomText, MAGic
  • Alternative Input: Switch control, eye tracking

Ongoing Efforts

We continuously monitor and improve accessibility through:

  • Regular automated accessibility testing (axe, WAVE, Lighthouse)
  • Manual testing with assistive technologies
  • User testing with people with disabilities
  • Third-party accessibility audits (annual)
  • Staff accessibility training programs
  • Dedicated accessibility team

Known Limitations

We are actively working to address the following accessibility limitations:

  • Real-time monitoring dashboards: Working to improve ARIA live region announcements for dynamic vital sign updates
  • Quantum visualization graphics: Developing text-based alternatives for quantum circuit diagrams
  • PDF reports: Adding tagged PDF generation for screen reader compatibility

Expected resolution: Q1 2025

Accessibility Standards & Regulations

Lydian Agent strives to meet or exceed these accessibility standards:

  • WCAG 2.1 Level AA: International web accessibility standard
  • Section 508: U.S. federal accessibility requirements
  • ADA Title III: Americans with Disabilities Act compliance
